## [0.1.0] - 2026-07-16

### Added

- First supported Atlas Download Manager release for macOS and Linux.
- Homebrew installation through `xkam7ar/tap/atlas-download-manager`.
- Plan-first CLI and interactive menu for video, audio, files, batches,
  website mirrors, and open directories.
- Public contribution, security, governance, and dependency-compliance
  documentation.
- Reproducible lock-derived SBOM and runtime-license audit instructions.
